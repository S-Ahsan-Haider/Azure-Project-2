- Gunicorn (short for Green Unicorn) is a Python WSGI HTTP server 
  used to run Python web applications in production. 
  It acts as a bridge between your Python app (Flask, Django, FastAPI, etc.) 
  and the web server, handling requests efficiently with multiple worker processes.
